## :octocat:Clone this repo

1. Clone :

```sh
  $ git clone https://github.com/VictorGabrielMS/uiclone_facebook_postpage.git
```

2. Execute:

```sh
  $ cd uiclone_facebook_postpage
```

3. Execute:

```sh
  $ npm install
  $ npm run start
```
